Задача 1

Спросите имя пользователя с помощью методы prompt. Выведите с помощью alert сообщение 'Ваше имя %имя%'.

Задача 2

Создайте переменную, присвойте ей число. Возведите это число в квадрат. Выведите его в консоль.

Задача 3

Определить является ли введенное число а кратным введенному числу b. (остаток от деления % равен 0). Обе переменные вводим с клавиатуры, ответ вывести в консоль.

Задача 4

Вывести в консоль кубы чисел от 3 до n с шагом 3. n вводим с клавиатуры. (циклы)

Задача 5

Вводим с клавиатуры 5 чисел (вводим числа с клавиатуры внутри цикла), как отрицательные, так и положительные. Ответом должна быть сумма только положительных чисел (отрицательные в сумму не должны входить).
